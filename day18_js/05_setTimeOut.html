window 메서드 중 실행 시간을 지정 메서드

  • setTimeout(function, millseconds); 일정시간 후에 function을 실행 시킬때 사용
  • setTimeout을 할당한 객체를 clearTimeout(할당한 객체) 취소 가능
  • setInterval(callback function, millseconds); 일정 간격으로 지속적 함수를 호출할 때 사용
  • clearInterval(할당받은 객체) 중지가능

millisecond : 1/1000 => 1초 1000 / 0.5초 500
