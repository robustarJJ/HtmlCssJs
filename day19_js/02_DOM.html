DOM(Document Object Model) : 문서(html) 객체 모델

  • '' 하위 태그는 각각의 기능(method), 속성(properties)을 가지는 문서 객체이다
  • 엘리먼츠 가져오기 : tag element로 가져오는 메서드 byId, by className, byTagName
  • document.getElementById('idName') : 단일 엘리먼츠 리턴
  • document.getElementByClassName('className') : 복수 엘리먼츠 리턴 => HTML Collection 타입으로 리턴
  • document.getElementByTagName('tagName') : 복수 엘리먼츠 리턴 => HTML Collection 타입으로 리턴
