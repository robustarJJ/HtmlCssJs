정규식(Regular Expression) : 문자열에서 특정 데이터를 탐색, 확인할 때 사용

  • RegExp 단어로 많이 표현됨.
  • 특정 데이터를 탐색, 확인, 존재여부 등을 쉽게 확인하기 위해 만든 규칙
  • 웹 분야에서는 password작성 규칙, 첨부파일 등에서 사용
  • 알고 있으면 편리한 기능. 다른 언어에서도 거의 비슷한 형식으로 사용
  • Regexp/flag : (//) : 시작, 종료기호, regexp : pattern, flag : 규칙옵션
  • ("str").match(/regexp/flag) : str에서 매칭되는 항목들을 객체(배열과 유사)로 반환
  • ("str").replace(/regexp/, "바꿀문자열") : 정규 표현식에 매칭되는 글자들을 바꿀문자열로 반환
  • ("str").split(/regexp/) : 문자열을 정규표현식에 매칭되는 항목으로 분리하여 배열로 반환
  • (/regexp/).test("str") : 문자열이 정규 표현식 규칙에 맞으면 true, 아니면 false
  • 정규표현식 플래그
  • i : 대/소문자 구분x
  • g : 전체 검색
  • m : 행바꿈도 포함
  • a-zA-z : 영어 알파벳 / ㄱ-ㅎ가-힣 : 한글 / 0-9 : 숫자
  • . : 모든문자열(숫자, 한글, 영어, 특수문자, 공백 모두) 줄바꿈x
  • \d : 숫자 \D : 숫자 아닌것 \s : 공백 \S : 공백 아닌것 \w : 영숫자 \W : 영숫자 아닌것 \특수기호
  • 검색패턴 : | or 패턴 a|b
  • [ ]: 괄호안의 문자들 중 하나 (or 처리의 묶음) /abc/ => "abc"를 포함하는 /[abc]/ => a또는 b또는 c를 포함하는 /[다-바]/ => 다, 라, 마, 바
  • [^문자] : 괄호안의 문제 빼고 [^abcd] : "a","b","c","d" 4개 제외
  • ^ 문자열$ : ^(시작) $(종료)
